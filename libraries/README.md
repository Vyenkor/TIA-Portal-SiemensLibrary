# Expanded Libraries

This folder contains expanded TIA Portal global library folders.

Keep each library folder complete. TIA Portal may require internal folders such as:

- `AdditionalFiles`
- `IM`
- `src`
- `System`
- `TMP`
- `UserFiles`
- `Vci`
- `XRef`

Do not remove these folders if you want the `.al21` library to remain openable.
